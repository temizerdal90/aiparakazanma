Prompt writing

Learn how to write AI prompts that give ChatGPT clear context.

A good prompt is not magic. It is a clear brief: role, goal, audience, constraints, examples and the format you want back.

Role

Set the lens

Example: "Act as a beginner-friendly content strategist."

Goal

Name the task

Example: "Create a 7-day content plan for Instagram Reels."

Context

Add details

Audience, topic, platform, tone, skill level and what to avoid all matter.

Output format

Make it usable

Ask for a table, checklist, script, outline or numbered plan.

Weak prompt

Give me content ideas.

Better prompt

Act as a beginner-friendly content strategist. Give me 15 Instagram Reels ideas for new freelancers learning AI tools. Use a table with hook, lesson, visual idea and CTA. Avoid income promises.

Reusable formula

Act as [role]. Help me [goal] for [audience]. Context: [details]. Output as [format]. Avoid [constraints].
